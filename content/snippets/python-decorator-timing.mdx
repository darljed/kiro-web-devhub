## Overview

A timing decorator that tracks execution duration, accumulates statistics (min, max, avg, p95), alerts on slow executions, and optionally emits metrics to external systems. Works with both sync and async functions.

## Code

```python
@timed(slow_threshold_ms=100)
def slow_operation():
    ...

with timer("block_name"):
    ...
```

## Notes

- `time.perf_counter()` provides the highest resolution timer available on the platform
- The stats registry enables a `/metrics` endpoint to expose all timing data
- `slow_threshold_ms` triggers WARNING-level logs for performance regression detection
- The `emit_metric` callback integrates with DataDog, Prometheus, or StatsD
- Supports both sync and async functions via `asyncio.iscoroutinefunction` detection
