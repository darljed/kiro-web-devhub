        content {

        }
      }

        for_each = var.secrets
        content {

          value_source {

          }
        }
      }

        http_get {

## Overview

A production Cloud Run service with VPC connectivity for private database access, Secret Manager integration, health probes, startup CPU boost for cold starts, and custom domain mapping.

## Code

```hcl
module "api" {
  source       = "./modules/cloud-run"
  service_name = "api"
  image        = "gcr.io/project/api:v1.0"
  min_instances = 1
  env_vars     = { DB_HOST = "10.0.0.5" }
  secrets      = { DB_PASSWORD = "db-password-secret" }
}
```

## Notes

- `min_instances = 0` enables scale-to-zero (cost saving); set > 0 for low-latency requirements
- `cpu_idle = true` throttles CPU between requests (cheaper but adds latency)
- `startup_cpu_boost` allocates extra CPU during cold start for faster initialization
- VPC connector enables access to Cloud SQL, Memorystore, and other VPC resources
- Secret Manager integration avoids passing secrets as plain environment variables
