## Overview

Advanced pivot table techniques for multi-dimensional business data analysis. Covers revenue breakdowns, month-over-month growth calculations, ranked product performance, and cohort retention analysis using pandas pivot tables and groupby operations.

## Code

```python
pivot = pd.pivot_table(

    values="revenue",
    index="region",
    columns="quarter",
    aggfunc="sum",
    margins=True,
)
```

## Notes

- `margins=True` adds row/column totals automatically
- Multi-value aggregation supports different functions per metric via dict `aggfunc`
- `pct_change()` on a pivoted DataFrame gives period-over-period growth rates
- Ranking within groups uses `groupby().rank()` on the pivot result
- For very large datasets, consider `dask` DataFrames which support the same pivot API
