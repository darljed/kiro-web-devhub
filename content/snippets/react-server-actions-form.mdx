## Overview

A form pattern using React Server Actions with progressive enhancement. The form works without JavaScript (submits to the server action directly), but with JS enabled provides inline validation, loading states, and optimistic feedback using `useActionState`.

## Code

```tsx
const [state, formAction, isPending] = useActionState(submitContact, initialState);

<form action={formAction}>
  <input name="email" aria-invalid={!!state.errors?.email} />
  <button disabled={isPending}>Submit</button>
</form>
```

## Notes

- `useActionState` replaces the older `useFormState` API in React 19+
- Server-side Zod validation ensures data integrity even if client validation is bypassed
- ARIA attributes (`aria-invalid`, `aria-describedby`, `role="alert"`) ensure screen reader accessibility
- Works without JavaScript via progressive enhancement (form submits natively)
- The `isPending` state enables loading indicators without separate loading state management
