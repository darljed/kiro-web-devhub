## Overview

A complete fluid typography system that smoothly scales text and spacing between viewport sizes without media query breakpoints. Using CSS `clamp()` with custom properties creates a mathematically consistent type scale that respects user preferences and works across all screen sizes.

Key design decisions:
- **clamp() function** provides min, preferred, and max values in a single declaration
- **Modular scale** uses different ratios at mobile vs desktop for optical harmony
- **Custom properties** make the system configurable and composable
- **Fluid spacing** pairs with fluid type to maintain vertical rhythm at all sizes
- **ch-based measure** limits line length for readability regardless of font size
- **text-wrap: balance** prevents orphaned words in headings

## Notes

- Use [Utopia](https://utopia.fyi) to generate precise clamp() values for your specific min/max viewport and type scale
- The `calc(Xvw + Yrem)` preferred value in clamp() creates a linear interpolation between min and max sizes
- Setting `html { font-size: 100% }` respects the user's browser font size preference
- `text-wrap: balance` is supported in Chrome 114+ and Safari 17.4+; use as progressive enhancement
- Pair fluid type with a CSS reset that removes default margins to fully control vertical rhythm
- For dark mode, consider slightly increasing font-weight or letter-spacing for legibility
- Test with browser zoom (200%) to ensure text remains readable at all zoom levels
