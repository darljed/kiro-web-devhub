## Overview

FastAPI's dependency injection system enables clean separation of concerns for database access, authentication, authorization, and service layers. Dependencies are composable and hierarchical, with automatic lifecycle management for resources like database sessions.

## Code

```python
DB = Annotated[AsyncSession, Depends(get_db)]
Auth = Annotated[CurrentUser, Depends(get_current_user)]

@app.get("/users/me")
async def get_me(user: Auth, db: DB):
    ...
```

## Notes

- `Annotated` type aliases keep route signatures clean while preserving type information
- Database sessions auto-commit on success and rollback on exception via the generator pattern
- `require_roles` is a dependency factory that creates parameterized access control checks
- Dependencies are resolved once per request and shared when reused (singleton per request)
- FastAPI runs async dependencies concurrently when they do not depend on each other
